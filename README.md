# Orrery

A 3D galaxy where every past meeting is a glowing sphere, clustered by meaning. Ask a question and the camera flies you to the answer. Navigation is retrieval.

## Quick Start

```bash
npm install
npm run dev
```

## With Claude API (optional)

Set `ANTHROPIC_API_KEY` for real-time LLM-powered answer synthesis:

```bash
ANTHROPIC_API_KEY=sk-ant-... npm run dev
```

Without the key, Orrery falls back to pre-baked pattern-matched answers.

## Stack

- React Three Fiber + drei + postprocessing
- Custom GLSL shaders (Fresnel rim, additive bloom)
- TF-IDF search (384-dim vectors, brute-force cosine similarity)
- Claude API (claude-haiku-4-5) for answer phrasing
- Web Speech API for voice input
- Zustand for state management

## Data

138 synthetic meetings across 10 topic clusters with 3038 moments. Generated by `scripts/generate-data.mjs`.
